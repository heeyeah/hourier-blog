## 안녕하세요!

Hourier Blog에 오신 것을 환영합니다.

이 블로그는 **개발과 일상을 기록하는 공간**입니다. 주로 다음과 같은 내용을 다룰 예정이에요.

- 개발하면서 배운 것들
- 기술적인 고민과 해결 과정
- 가끔은 일상 이야기도

## 기술 스택

이 블로그는 아래 기술로 만들어졌습니다.

```
Next.js + Tailwind CSS + Contentlayer
```

폰트는 **Pretendard**를 본문에, 코드 블록에는 **JetBrains Mono**를 사용하고 있습니다.
